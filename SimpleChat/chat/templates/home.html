SimpleChat

SimpleChat is a very very basic chat engine that should work perfectly on just about any device with a "web browser". I very specifically don't use any javascript or fancy styling.

There's even an API of sorts. Hit {% url 'json_new' %} to create a new chat with JSON feedback. Then POST to [party_a_url] with a form variable called "lines" with the message you wish to send. You'll get back a JSON response with the full chat log.

Feel free to send feedback to arbitraryuser@gmail.com
